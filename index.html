Cross-device sync

Syncs your holdings and trade log across devices using a private GitHub Gist as storage — nothing else touches it. On your first device, paste a GitHub token and leave Gist ID blank to create one. On every other device, paste the same token plus the Gist ID it gives you.

Create a token at github.com/settings/tokens → Generate new token (classic) → check only the gist scope. Keep it private; anyone with it can read/write your gists.
